## Issue ドキュメントの役割

このディレクトリの Issue ドキュメントは、以下の目的で使用されます：

- **仕様の詳細化**: GitHub Issue では記述しきれない詳細な仕様
- **技術的決定の記録**: アーキテクチャや設計の決定事項
- **進捗の追跡**: 開発状況や課題の記録

## 注意事項

- **GitHub Issue との併用**: このドキュメントは GitHub Issue の補助資料として使用してください
- **コミット**: Issue の変更や進捗状況は適宜 Git コミットしてください
- **同期**: GitHub Issue のステータスを正として、ドキュメントを参考資料として扱ってください
